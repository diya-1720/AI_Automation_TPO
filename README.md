# SPC Assistant - AI Documentation Platform

![Project Status](https://img.shields.io/badge/Status-Active_Development-indigo) ![Version](https://img.shields.io/badge/Version-v1.0.0-blue) ![License](https://img.shields.io/badge/License-MIT-green)

An intelligent, AI-powered document generation and report automation platform designed for college **Training & Placement Cells (TPO)**, academic committees, and institutional departments.

SPC Assistant automates the end-to-end event documentation workflow: processing raw event evidence (handwritten notes, scanned papers, images, PDFs, DOCX), performing multi-modal OCR text extraction via Gemini Vision, scoring field confidence, embedding event notice flyers, arranging multi-photo event grids into Word tables, and producing standardized, publication-ready DOCX and PDF reports.

---

## 🌟 Key Features

### 1. 👥 Shared Team Account Authentication
- **Team Collaboration Model**: Designed for team collaboration across 20–30 members using single/shared credentials.
- **Global Document Access**: Reports generated by any team member persist globally in `generated/reports_db.json` and are instantly accessible in **Previous Reports**, preventing duplicate report creation.
- **Default Shared Credentials**: Automatically seeds default shared account (`spc@tpo.edu` / `spc12345`) for team access.

### 2. ⚡ AI Auto-Fill & Gemini Vision OCR
- **Handwritten & Printed OCR**: Upload photos of handwritten meeting notes, scanned agendas, or transcripts.
- **Automated Field Population**: Gemini Vision API extracts key activity details (*Activity Name, Date, Venue, Department, Organizer, Coordinator, Resource Person, Objectives, Outcomes, Description*) and populates form fields automatically.

### 3. 📄 Notice & Brochure Dedicated Upload
- Dedicated file upload area for Notice, Brochure, or Event Announcement flyers.
- Automatically embeds notice images directly into designated sections of output documents with aspect ratio preserved.

### 4. 🖼️ Event Photos Section (Multiple Uploads & 2-Column Grid Layout)
- **Multi-Photo Selection**: Upload multiple event photographs at once (2, 4, 6, 8+ photos) with thumbnail previews and deletion controls.
- **Automated Word Table Grid**: The backend engine arranges all uploaded event photos into a structured 2-column Word table grid on a dedicated page ("Event Photographs"), preserving aspect ratio and preventing chaotic layout overflow.

### 5. ☑️ Consolidated 3-State Checklist Grid
- Replaces individual text inputs with a structured 3-state toggle selector:
  - **✔ Yes** (Checked)
  - **✖ No** (Unchecked)
  - **Blank** (Unset / Optional)

### 6. 📁 Previous Reports & Strict Standardized Naming
- Automatically saves all generated documents into `generated/reports_db.json`.
- Enforces strict dynamic file naming: `[Name_of_the_Activity]_documentation.pdf` / `.docx`.
- Built-in **Previous Reports** viewer with search, preview, and dual export downloads.

### 7. ⚙️ Platform Settings & Global Dark Mode
- **Profile**: Set shared Account Name and associated Email.
- **Appearance**: Toggle switch for **Dark Mode** and **Light Mode** applied globally.
- **Default Auto-Fill Data**: Save default institutional parameters (*Default College Name, Department, Organizer, Venue*) that automatically pre-fill the **New Report** form.
- **Export Preferences**: Configure default download format (*Always download as PDF*, *Always download as DOCX*, or *Ask every time*).

### 8. 🎨 4-Step AI Template Studio (PDF & DOCX)
- Drag-and-drop support for master template documents (DOCX or PDF).
- Confidence scores (`0-100%`) with visual warning badges for low-confidence fields.
- Synthesizes professional academic summaries (Activity Summary, Objectives, Outcomes, Event Description).

## 🛠️ Technology Stack

- **Frontend**: React 18, Vite, TypeScript, Tailwind CSS, Lucide Icons, React Router.
- **Backend**: FastAPI, Python 3.10+, Uvicorn, `python-docx`, `pypdf`, Pillow (`PIL`), `docx2pdf`, Google Generative AI (Gemini 1.5 Flash).
- **Data Persistence**: JSON Document Stores (`users_db.json`, `settings_db.json`, `reports_db.json`).

## 🚀 Setup & Installation

### Prerequisites
- Python 3.10 or higher
- Node.js 18 or higher
- Gemini API Key

### 1. Backend Setup
```bash
cd backend

# Create & activate virtual environment (Windows)
python -m venv venv
.\venv\Scripts\activate

# Install dependencies
pip install fastapi uvicorn python-docx docx2pdf python-dotenv google-generativeai pydantic pypdf pillow

# Create .env file
echo GEMINI_API_KEY=your_gemini_api_key_here > .env

# Start FastAPI backend server
python -m uvicorn main:app --reload --port 8000
```
*(Backend running on `http://localhost:8000`)*

### 2. Frontend Setup
```bash
cd frontend

# Install npm packages
npm install

# Start Vite development server
npm run dev
```
*(Frontend running on `http://localhost:5173`)*

## 📖 User Workflow

1. **Sign In**: Log in using the shared team account (`spc@tpo.edu` / `spc12345`).
2. **New Report**:
   - Pre-fills default institutional data from Settings.
   - Upload handwritten notes / images to **AI Auto-Fill & Vision OCR** to auto-populate form fields.
   - Upload Notice & Brochure image.
   - Select multiple **Event Photos** (automatically formatted into a 2-column Word table grid).
   - Set 3-state checklist values (✔, ✖, Blank).
   - Click **Generate Document**.
3. **Previous Reports**: Access, search, preview, and download all finalized team reports saved in `[Name_of_the_Activity]_documentation.pdf` format.
4. **Templates**: Upload custom master templates (DOCX / PDF) and analyze placeholder mappings.
5. **Settings**: Customize default values, toggle Dark/Light mode, and set export file preferences.
